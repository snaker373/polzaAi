# Polza.ai — статическая копия лендинга

Полная статическая копия главной страницы [polza.ai](https://polza.ai/), сохранённая для локального/учебного использования.

## Содержимое

- `site/index.html` — серверно-отрендеренная разметка главной страницы
- `site/_next/` — CSS и JS-чанки (Next.js)
- `site/lendos/` — изображения, иконки, видео и GIF
- `site/InterVariable.woff2`, `site/Logo.svg` — шрифт и логотип

Ссылки на оптимизатор изображений Next.js (`/_next/image?...`) переписаны на
исходные локальные файлы, чтобы копия работала полностью офлайн.

## Запуск локально

```bash
cd site
python -m http.server 8900
# открыть http://127.0.0.1:8900/
```

## Примечание

Контент, бренд и материалы принадлежат правообладателям polza.ai.
Копия сделана исключительно для ознакомления и не предназначена для публикации.
